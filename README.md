**Installation**:


```

npm install -g bower

```
After install bower. Run command below to get NodeJS libraries.

```

npm install

```
and run

```
bower install
```
**Restore database**: open cmd, cd to db folder and run below command
```

mongorestore

```


Run `node server` and see it in **http://localhost:3000**
